# Code of Conduct

## Our Pledge

We pledge to make participation in this project and community a harassment‑free experience for everyone, regardless of level of experience, gender, age, race, or any other personal characteristic.

## Our Standards

Examples of **positive** behavior:

- Using welcoming and inclusive language
- Being respectful of different viewpoints and experiences
- Giving and accepting constructive feedback
- Focusing on what is best for the community

Examples of **unacceptable** behavior:

- Harassing comments or personal attacks
- Trolling or insulting/derogatory remarks
- Publishing others' private information without permission
- Any behavior that would be considered inappropriate in a professional setting

## Responsibilities

Project maintainers are responsible for clarifying standards of acceptable behavior and are expected to take corrective action in response to any unacceptable behavior.

They have the right to remove, edit, or reject comments, commits, code, and other contributions that do not align with this Code of Conduct.

## Scope

This Code of Conduct applies within all project spaces, including:

- GitHub issues and pull requests
- Discussions in social channels or community spaces related to the project

## Enforcement

If you experience or witness behavior that violates this Code of Conduct, please contact the maintainer at the email address listed in the repository README.

All good‑faith reports will be reviewed and investigated as needed.
